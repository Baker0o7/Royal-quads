# Royal Quad Bikes — Flutter App

Quad bike rental management system for Mambrui Sand Dunes, Kenya.
Rebuilt from TypeScript/React/Capacitor to pure Flutter/Dart.

## Features
- Booking with live countdown timer & overtime billing
- M-Pesa payment flow with confirmation code
- Admin dashboard with PIN lock & quick-start
- PDF receipt export & WhatsApp sharing
- Pre-bookings, waitlist, promo codes
- Safety waiver, staff management, maintenance logs
- Google Sign-In

## Build
```bash
flutter pub get
flutter build apk --debug
```
