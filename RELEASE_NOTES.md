## 🏍️ Royal Quads v4.9.6 - Complete Quad Bike Rental System

### New in v4.9.6:
- **Quick Start Multi-Quad**: Start unlimited rides at once with per-quad guide names, payment methods (Cash/M-Pesa/Shee) and live 20% commission breakdown
- **Pause & Extend Rides**: Pause active rides and resume later. Extend rides with +5/10/15/30 min quick-select options
- **Enhanced Admin Fleet**: Expandable quad cards with total rides, revenue (KES) and minutes stats per quad. IMEI displayed, delete with confirmation
- **History Search & Sort**: Search by name/phone/quad/receipt, date filter, sort by newest/oldest/highest/lowest, date group headers, CSV export button
- **9-Clause Safety Waiver**: Comprehensive waiver with 9 clauses (inherent risks, health confirmation, equipment, alcohol policy, financial liability, release & indemnity, minors), scroll-to-read enforcement and risk banner with pills
- **QR Code Receipts**: Every receipt now includes a QR code for quick verification with perforated divider design
- **Pre-book Notes & M-Pesa Ref**: Add special requests and M-Pesa reference when pre-booking a ride
- **Bug Fixes**: All unprotected `.then()` calls in Admin.tsx now have `.catch()` error handlers, clipboard operations wrapped in try/catch, localStorage safety throughout

### Features:
1. User-friendly interface for booking quad bikes.
2. Admin panel for managing bookings and vehicles.
3. Notifications for bookings and cancellations.
4. Integration of payment gateways for seamless transactions.
